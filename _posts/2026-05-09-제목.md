오늘 GitHub Pages로 작은 웹 공간을 만들었습니다.

내 블로그 주소: https://username.github.io

처음에는 저장소, commit, push 같은 말이 전부 낯설었지만, 해보고 나니 각각은 하나의 사건에 가까웠습니다.

- commit은 지금까지 바꾼 내용을 저장점으로 남기는 일입니다.
- push는 내 작업을 GitHub로 보내는 일입니다.
- GitHub Pages는 저장소 안의 파일을 웹사이트 주소로 보여주는 기능입니다.

아직은 글 하나뿐이지만, 이제 캡처가 아니라 직접 열어볼 수 있는 주소가 생겼습니다.

**엑스 리브리스(Ex Libris) 는 라틴어로 "~의 책 중 하나" 또는 "~의 서재에서"라는 뜻으로, 과거 저자가 남의 책을 인용할 때 사용하던 관용구입니다. 또한, 책 소유권을 표시하기 위해 책표지(book-plate)에 붙이던 표식을 의미하기도 합니다.**
